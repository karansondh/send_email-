# send_email-
using this peice of code send you'r emails easily using python add some details by yourself and you are good to go!
## Follow the steps below in order to run the program:-
* Download the code as zip file, extract it or one can copy too.
* Then in your IDE edit email_sender.py.
* fill in the details which are required otherwise email will not be sent.
* Then go in index.html file and add the body of you'r email in between the body tags.
* Thats it!! run the file and you'r console/terminal should say all good boss!, means the email has been sent.
* sometimes due to the subject gmail automatically sends the emails in the junk folder so add a good subject to the email.
